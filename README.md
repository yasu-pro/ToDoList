# Features

"TodoList"として、ソート機能、タスク状況が確認できるツールを作成した。

# Requirement

- node 20.5.0
- npm 9.8.1

# Usage

```zsh
git clone git@github.com:yasu-pro/ToDoList.git
cd ToDoList-app
npm install
npm run build
npm run start
```
